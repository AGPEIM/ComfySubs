# ComfySubs

> 基于electron-vue的构建的字幕添加工具

## 应用说明
**⚠️当前应用仍在开发阶段**

本项目致力于开发一个跨平台的、轻量级视频字幕添加工具。借助开放的语音识别接口，实现Ai添加字幕。
